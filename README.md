# Dataportal Backend

## Goal

The Dataportal Backend provides the necessary functionality for the corresponding [Dataportal UI](https://github.com/medizininformatik-initiative/dataportal-ui).
The main feature is the management and dispatching of feasibility queries ([CCDL](https://github.com/medizininformatik-initiative/clinical-cohort-definition-language))
and dataqueries ([CRTDL](https://github.com/medizininformatik-initiative/clinical-resource-transfer-definition-language)).

For further information check the docs page of [this project](https://medizininformatik-initiative.github.io/dataportal-backend/)
as well as the one of the [FDPG Dataportal](https://medizininformatik-initiative.github.io/dataportal/).
